Titular de la web

Assessoria d'Habitatge · Dades del titular pendents de verificació abans de publicació definitiva

Objecte

Aquesta web ofereix eines orientatives per analitzar el mercat de l'habitatge a Catalunya. Les recomanacions no constitueixen assessorament professional vinculant.

Propietat intel·lectual

Els continguts propis d'aquesta web estan protegits per la normativa aplicable. També poden existir marques, logos o continguts de tercers amb els seus respectius drets.

Limitació de responsabilitat

Les eines i dades proporcionades tenen caràcter informatiu i orientatiu. Assessoria d'Habitatge no es responsabilitza de decisions preses en base a la informació d'aquesta web. Recomanem consultar professionals qualificats.

Legislació aplicable

Aquesta web es regeix per la legislació espanyola. Per a qualsevol controvèrsia, les parts se sotmeten als jutjats de Tarragona.
